# CR Consig - 1.0

## Sem Kamal
### Dependências
  
  - Ruby 3.1.2
  - Rails 7.0.4
  - NPM > 7.1

Após instalar o Ruby com o gerenciador de versões de preferência instale as gems:

```console
bundle install
```

Crie o Banco de Dados:

```console
rails db:create db:migrate
```

Rode o seeds:

```console
rails db:seed
```

Verifique se já tem o npm e depois rode:

```console
npm install
```

Suba o servidor:

```console
bin/dev
```
